무수한 콜백으로 인한 복잡함
fetch(url).then((data)=>{return data.json()}).then((json)=>{console.log(json)})

간단하게 기다리게 할 수 있음. await을 사용하면 비동기로 동작하는 것들의
작업이 완료될때까지 기다린다.
사용하려면 사용하는 곳이 async로 선언이 되어야함
async function name(){ 
    const data = await fetch()
    const json = await data.json()
    console.log(json)
}
